# Planner Pro

할 일을 효율적으로 관리할 수 있는 Todo 관리 애플리케이션입니다.

## 🚀 배포 링크

[Planner Pro 바로가기](https://todo2026-f265.vercel.app/)

## 📋 주요 기능

- ✅ 할 일 추가/수정/삭제
- 🔍 검색 기능
- 🎯 우선순위 설정 (HIGH, MEDIUM, LOW)
- 📅 마감일 관리
- 🏷️ 카테고리별 분류
- 💾 localStorage를 통한 데이터 저장

## 🛠️ 사용 기술

- **Frontend**: React, TypeScript
- **Build Tool**: Vite
- **Styling**: CSS
- **Deployment**: Vercel

## 📦 설치 및 실행

```bash
# 저장소 클론
git clone https://github.com/CHPark00-world/todo2026.git

# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

## 💡 개발 과정

- React Hooks를 활용한 상태 관리
- TypeScript로 타입 안정성 확보
- localStorage를 통한 데이터 영속성 구현
- 반응형 디자인 적용
